Linear Regressions

Linear Regression is a predictive analysis method to explain the relationship between one dependent variable and one (or more) independent variable(s). This model is defined by formula y = c + b*x, where y = estimated dependent variable, c = constant, b = regression coefficient, and x = independent variable.

Simple Linear Regressions
