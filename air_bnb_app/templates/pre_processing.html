Our original ‘listings.csv’ had 16 columns with categorical and numerical data. We began cleaning our dataset using Python Pandas. We dropped rows that contained a $0 price per night and saved the updated dataset as ‘clean_listings.csv’. We next added one-hot encoding to categorical factors to ‘clean_listings.csv’ and saved a new file ‘ABB_ML_ready.csv’. We then ran a Seaborn correlation matrix, heatmap, and describe-command at 50%, 80%, and 90% to determine which factors could influence final prices.

We dropped categorical factors that provided no influence and concluded using 90% of the data would provide the best model accuracy. We renamed this dataset ‘abnb_df_90_ML.csv’ file to be used as our final dataset for ML algorithms.
