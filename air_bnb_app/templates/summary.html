Summary

(Gradient Boosting is bae.)

This project explored a sample Airbnb dataset with the goal of developing a price prediction model that is comparable to what the company is using.

Data on comma separated values were pre-processed to make it suitable for deeper analyses including processing it through different machine learning models.

Multiple technologies and tools were used in order to accomplish the goal. Including but limited to python, jupyter lab/notebook, flask, scikit learn, HTML and CSS.

After preparing the data, it was subjected to multiple machine learning regression models against pre-determined affecting categories. The resulting models' scores were then compared among each other to determine which is the best-fitting.

Once the best-fitting model was identified a front-end GUI was created with the chosen model at the backend to calculate the price based on the user's input.

Future Work

There are multiple avenues to expand this project:

  • Expand the sample dataset to cover a wider time range
  • Add a "host" and "renter" mode on the calculator
  • Add multiple regression models option on the calculator
  • Add different cities on the calculator
